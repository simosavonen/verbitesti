## Test Structure

### Easy Level (Helppo)
Abstract verbs requiring conceptual understanding:
- deserve, allow, propose, threaten
- praise, encourage, inform, protect
- achieve, develop, convince, remind

### Medium Level (Keskitaso)
More sophisticated vocabulary:
- mediate, deteriorate, advocate, suppress
- exaggerate, expand, consolidate, mitigate
- fluctuate, obliterate, tolerate, revoke
